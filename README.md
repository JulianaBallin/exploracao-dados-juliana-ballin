# Exploração de Dados Públicos com Estatística Descritiva

## Sobre o Projeto

Este repositório apresenta uma análise exploratória de dados (EDA – Exploratory Data Analysis) utilizando a Base de Dados da Carteira de Investimento do Ministério das Cidades, disponibilizada no portal oficial de Dados Abertos do Governo Brasileiro.

O objetivo do trabalho é aplicar técnicas de estatística descritiva e visualização de dados utilizando Python, com foco na compreensão da estrutura dos investimentos públicos voltados ao desenvolvimento urbano, habitação e infraestrutura.

---

## Dataset Utilizado

**Nome do Dataset:**  
Base de Dados da Carteira de Investimento do Ministério das Cidades

**Link Oficial:**  
https://dados.gov.br/dados/conjuntos-dados/base-de-dados-da-carteira-de-investimento-do-ministerio-das-cidades

**Órgão Responsável:**  
Ministério das Cidades – Governo Federal do Brasil

**Descrição Geral:**

Este conjunto de dados apresenta informações detalhadas sobre a carteira de investimento do Ministério das Cidades, consolidando dados de programas, investimentos e operações relacionados ao desenvolvimento urbano, habitação e infraestrutura.

A base foi construída a partir de fontes oficiais como:
- Transferegov
- BD Gestores
- Canal FGTS
- Minha Casa Minha Vida
- BNDES

O objetivo da disponibilização é promover transparência pública, facilitar o acompanhamento das políticas públicas e apoiar ações de fiscalização e controle social.

---

## Objetivos da Análise

- Realizar tratamento e limpeza dos dados
- Identificar valores ausentes
- Calcular estatísticas descritivas:
  - Média
  - Mediana
  - Moda
  - Mínimo e máximo
  - Desvio padrão
  - Variância
  - Quartis e intervalo interquartil (IQR)
- Gerar visualizações gráficas:
  - Histogramas
  - Boxplots
  - Gráficos de dispersão
- Identificar padrões, tendências e possíveis inconsistências

## Motivação da Escolha do Dataset

O dataset foi escolhido por abordar investimentos públicos em infraestrutura urbana e habitação, temas estratégicos para o desenvolvimento social e econômico do país. A análise permite compreender como os recursos públicos estão distribuídos entre programas e regiões, contribuindo para maior transparência e entendimento das políticas públicas.

---

## Conclusão

A análise exploratória permite identificar padrões de distribuição de investimentos, possíveis concentrações de recursos e variações significativas entre programas e operações. O estudo demonstra como técnicas de ciência de dados podem contribuir para análise de políticas públicas e transparência governamental.
